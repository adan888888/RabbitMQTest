# 🚀 RabbitMQ 超简单演示

## 什么是RabbitMQ？
RabbitMQ就是一个**消息队列**，就像邮局一样：
- **发布者** = 发邮件的人
- **队列** = 邮箱
- **订阅者** = 收邮件的人

## 🎯 这个演示做什么？
1. **发布者**发送3条消息到3个不同的队列
2. **订阅者**同时监听这3个队列
3. 你会看到消息是如何传递的

## ⚡ 一键运行
```bash
cd simple
./run_demo.sh
```

## 💡 核心概念

### 发布者 (Publisher)
```go
// 发送消息到队列
rabbit.SendMessage("orders", message)
```

### 订阅者 (Subscriber)
```go
// 监听队列，处理消息
rabbit.ReceiveMessages("orders", func(body []byte) {
    // 处理消息
})
```

## 🎉 就这么简单！

这就是RabbitMQ的基本用法：
- ✅ 发送消息
- ✅ 接收消息
- ✅ 队列管理
- ✅ 系统解耦

现在你理解了消息队列的核心概念！
